# Smart QR Code Generator (QRForge)

A modern, dynamic, and premium QR code generator built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Multiple Supported Types**: Generate QR codes for URLs, Text, Emails, Phones, SMS, Wi-Fi, vCards, Locations, UPI Payments, Social accounts, YouTube links, app links, and files.
- **Dynamic Updates**: Generate static or dynamic (trackable) QR Codes.
- **Scanner Integration**: Scan existing QR codes directly from your device camera or uploaded images.
- **Customization Options**:
  - Gradient colors and dot patterns (Square, Round, Dots).
  - Add logos to the center of your codes.
  - Granular error correction levels (L, M, Q, H).
- **Persistent Storage**: Uses local storage to keep track of your `savedQRs` and `scanHistory`, plus an analytics dashboard to review your total, unique, and daily scans.
- **Apple/Glassmorphism UI**: Beautifully crafted interface, clean typography and smooth micro-animations.

## How to use

1. Clone the repository or simply download the `index.html` file.
2. Open `index.html` in your favorite modern browser. No build steps or servers are required for local testing!
3. Login using any testing credentials as data securely persists to your browser's local storage.
